# Screenshot Slots

Place product screenshots in this folder using the names below so README sections can reference stable paths.

Recommended files:

- main-workspace.png
- ssh-host-picker.png
- file-manager-editor.png
- settings-and-palette.png

Image guidance:

- Use PNG format.
- Capture full UI context rather than cropped widgets.
- Prefer 16:9 or 16:10 shots with readable text.
- Avoid sensitive hostnames, credentials, or private filesystem paths in screenshots.
