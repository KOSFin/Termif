Privacy

This static website does not include analytics, advertising scripts, or tracking cookies. It requests public GitHub Releases data from the GitHub API so download buttons can point at current release assets.

The Termif desktop app stores settings, host metadata, and workspace state locally in the app data directory. Snippets and bounded terminal scrollback snapshots are currently stored in local WebView storage.

Imported SSH private keys are not copied into Termif storage. Managed hosts can currently include an optional password in local host metadata, so avoid saving sensitive production passwords until OS keychain storage lands.

GitHub may process access logs when you visit the repository or download release assets. Review GitHub's own privacy terms for those services.
